Socket.io Chat App

A real-time chat application

Project Report

Introduction:

The Socket.io Chat App is a web-based chat application that allows users to communicate in real-time. This project utilizes Node.js, Express.js, and Socket.io to create a seamless chatting experience. The application provides features such as user authentication, real-time chat, an active user list, and typing status notifications.

Project Objectives

  • Real-time Chat: Implement a real-time chat feature that allows users to exchange messages instantly.
  • User Authentication: Assign unique user IDs to each participant for user authentication.
  • Active User List: Display a list of active users in the chat room.
  • Typing Notifications: Notify users when someone is typing a message.

Technologies Used

  • Node.js: Used as the server-side runtime environment.
  • Express.js: Created a web server to serve static files and handle routes.
  • Socket.io: Enabled real-time communication between the server and clients.
  • HTML/CSS: Designed the user interface and layout.
  • JavaScript: Wrote client-side and server-side code.
